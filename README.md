# GastroPilot

## What is GastroPilot?

GastroPilot is a modern, web-based restaurant management system for hospitality businesses. It centralizes table and reservation planning, guest and order management, billing, and analytics in one platform, helping operators streamline service workflows and manage capacity efficiently. See https://gastropilot.org for more information.
